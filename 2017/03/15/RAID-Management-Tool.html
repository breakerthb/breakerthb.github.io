发表于 2017-03-15

搭建RAID management tool的步骤

1.硬件要求

webserver没有太多要求，但是target server 要求是windows 7 32bit的，因为raid的虚拟driver是针对32位的，所以target server必须是32bit的才可以:

  • Website server:

    Windows 7, Windows 2008 Server (64 bit)

  • Target server:

    Windows 7 (32 bit)

Hardware requirements:
Samsung SSD: PM1725, 950pro and 960pro

nvme盘需要特殊的转接卡，问DCEE组借了2个转接卡，才识别出来NVME盘（没有装驱动），折腾了一早上，终于在陈总他们组的华硕的机子上，用windows 7 64bit的系统识别出来了两个nvme盘，结果还是以不能安装虚拟驱动结束。

2.安装和配置

2.1 系统装好了以后，要配置网络，以及FTP，Telenet的相关配置，具体可以参考 

MSG_Windows_RAID_Management_Tool_v0.1_GUI_User_Guide(Windows).docx

2.2 安装虚拟RAID驱动，在计算机中搜索Hdwwiz.exe，然后安装王洋之前写好的虚拟RAIDdriver，具体参考pdf:

How to Install Windows Virtual RAID Driver.pdf

2.3 web服务器

target server配置好了以后，就可以在web界面上尝试连接了，RAID_Management_Tool的login密码是admin，admin，差点忘记了，尝试了半天（其实userguide上也有写）。

如果上面两步都配置好了以后，按理说是可以正常连接，并list出盘的，如果弹出失败，可以去target server 的C盘的user/Administrator 下查看是否有RAIDMgt.ext，并尝试执行 -I，如果cli上有盘的话，那么可能是连接出现了问题，如果没有盘的话，可能是没有配置好telnet或者其它。。。
